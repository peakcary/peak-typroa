 AI生图调用流程详解

  1. 前端触发（pages/pay/pay.js）

  用户点击"开始生成"按钮 → 调用 startGenerate() 方法：

  // 关键参数准备
  const uploadImageUrl = this.data.uploadedFileData  // 用户上传的图片
  const select = { [stylesEq.postKey]: stylesEq.typeId }  // 选中的样式
  const workflowUuid = this.data.workflowUuid  // 工作流ID

  // 调用云函数
  wx.cloud.callFunction({
    name: "generateImage",
    data: {
      uploadImageUrl,        // 用户图片URL
      select,                // 样式参数
      workflowUuid,          // AI工作流UUID
      classType,             // 类型
      amount,                // 金额
      deviceId,              // 设备ID
      typeId,                // 板块种类 (typeId=1 是AI生图)
      isAI: true,            // 标记为AI任务
      bgImg                  // 背景图
    },
    success: result => {
      // 获得 generateUuid，跳转到支付/预览页面
      wx.navigateTo({
        url: `/pages/payNew/payNew?orderId=${result.result.out_trade_no}&uuid=${result.result.generateUuid}&isAI=true`
      });
    }
  });

  2. 云函数处理（generateImage/index.js）

  云函数收到请求后执行以下步骤：

  步骤 A：创建订单记录

  // 在数据库创建 orders 表记录
  await db.collection('orders').add({
    out_trade_no: newOrderId,
    userId: OPENID,
    uploadImageUrl,      // 保存用户图片URL
    generateUuid: null,  // 初始为空
    workflowUuid,        // AI工作流ID
    select,              // 选中的样式
    status: "pending",
    typeId,
    isAI: true
  });

  步骤 B：调用第三方AI生图API

  // 使用 liblibai 的 ComfyUI API
  const response = await axios.post(
    "https://openapi.liblibai.cloud/api/generate/comfyui/app",
    {
      templateUuid: "4df2efa0f18d46dc9758803e478eb51c",
      generateParams: {
        "133": {
          class_type: "LoadImage",
          inputs: {
            image: uploadImageUrl  // 👈 用户上传的图片
          }
        },
        "169": {
          class_type: classType,
          inputs: {
            ...select,             // 👈 样式参数
            typeId: Number(typeId)
          }
        },
        workflowUuid             // 👈 工作流配置
      }
    },
    {
      headers: { "Content-Type": "application/json" },
      params: {
        AccessKey: "hY8O7vieXUwED_jccKHe1A",
        Signature: signature,    // HMAC-SHA1签名
        Timestamp: timestamp,
        SignatureNonce: nonce
      }
    }
  );

  步骤 C：处理响应

  const { code, data } = response.data;

  if (code === 100054) {
    // 任务排队中，保存到 image_task_queue
    await db.collection('image_task_queue').add({
      out_trade_no: newOrderId,
      status: 'pending'
    });
    return { success: true, generateUuid: null, inQueue: true };
  }

  // 获取生成任务ID并更新订单
  const generateUuid = data.generateUuid;
  await db.collection('orders').where({ out_trade_no: newOrderId }).update({
    data: { generateUuid }
  });

  return {
    success: true,
    out_trade_no: newOrderId,
    generateUuid,
    inQueue: false
  };

  3. 查询生图状态（checkGenerateStatus/index.js）

  前端定时轮询查询生图结果：

  // 前端定时调用（通常在支付预览页面）
  wx.cloud.callFunction({
    name: "checkGenerateStatus",
    data: { generateUuid },
    success: result => {
      if (result.result.success) {
        // 获得生成的图片数组
        const images = result.result.images;  // 返回6张图片
        // 显示预览和合成
      }
    }
  });

  云函数查询逻辑：
  const response = await axios.post(
    "https://openapi.liblibai.cloud/api/generate/comfy/status",
    { generateUuid },
    { params: { AccessKey, Signature, Timestamp, SignatureNonce } }
  );

  const imagesData = response.data.data?.images || [];
  const percentCompleted = response.data.data?.percentCompleted || 0;

  // 返回前6张有效的图片
  return {
    success: true,
    images: imagesData.slice(0, 6),
    percentCompleted
  };

  4. 核心流程图

  用户上传图片
     ↓
  选择样式和设置
     ↓
  点击"开始生成" → 调用 generateImage 云函数
     ↓
     ├─ 创建订单（orders表）
     ├─ 调用 liblibai API 提交 ComfyUI 任务
     │  └─ 传递：用户图片 + 样式参数 + 工作流配置
     ├─ 获取 generateUuid（任务ID）
     └─ 更新订单的 generateUuid
     ↓
  返回 generateUuid 到前端
     ↓
  前端跳转到 payNew 页面
     ↓
  前端定时轮询 checkGenerateStatus（每 1-2 秒查询一次）
     ↓
  获取生成的图片数组（通常返回6张）
     ↓
  Canvas 合成最终图片
     ↓
  用户确认支付
     ↓
  生成订单并打印

  5. 关键参数说明

| 参数                  | 来源            | 说明                             |
| --------------------- | --------------- | -------------------------------- |
| uploadImageUrl        | 用户上传        | 云存储图片URL                    |
| workflowUuid          | 从index页面传入 | ComfyUI工作流ID                  |
| select                | 选择的样式      | 样式参数（如 {style_param: 1})   |
| templateUuid          | 硬编码          | 4df2efa0f18d46dc9758803e478eb51c |
| generateUuid          | API返回         | AI生图任务ID，用于查询进度       |
| ACCESS_KEY/SECRET_KEY | 硬编码          | liblibai API认证                 |

  6. 支持的工作流配置

  Python服务定义了5套工作流配置：
  - workflow_api_0.json - workflow_0
  - workflow_api_1.json - workflow_1
  - workflow_api_2.json - workflow_2
  - workflow_api_3.json - workflow_3
  - workflow_api_4.json - workflow_4

  通过 workflowUuid 参数选择使用哪套配置。

  7. 异常处理

  - 如果API返回 code === 100054：任务排队中，稍后重试
  - 如果没有 generateUuid：任务创建失败
  - 前端轮询时如果 percentCompleted < 1：任务处理中
