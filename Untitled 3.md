  📊 数据流现状

  当前: 各页面独立
  ├─ index: data() { bannerList, products }
  ├─ category: data() { searchKeyword, products }
  ├─ cart: data() { cartItems }
  ├─ profile: data() { userInfo }
  └─ ❌ 无跨页面通信，无共享状态

  建议: Pinia 全局状态
  ├─ store/cart.ts { items, total, addItem() }
  ├─ store/user.ts { userInfo, token, login() }
  ├─ store/product.ts { products, categories }
  └─ ✅ 所有页面共享，自动同步
