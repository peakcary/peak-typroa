3.1 接口规范说明

  基础URL：
  - 开发环境：http://localhost:3000/api
  - 生产环境：https://dzbxg88.cn/api

  通用请求头：
  Content-Type: application/json
  Authorization: Bearer {access_token}
  X-Request-ID: {unique_request_id}

  通用响应格式：
  {
    "code": 200,
    "message": "success",
    "data": {},
    "timestamp": 1705891200000
  }

  状态码说明：
  ┌────────┬─────────────────────────────┐
  │ 状态码 │            说明             │

  ├────────┼─────────────────────────────┤
  │ 200    │ 请求成功                    │
  ├────────┼─────────────────────────────┤
  │ 400    │ 请求参数错误                │
  ├────────┼─────────────────────────────┤
  │ 401    │ 未授权（未登录或token过期） │
  ├────────┼─────────────────────────────┤
  │ 403    │ 无权限访问                  │
  ├────────┼─────────────────────────────┤
  │ 404    │ 资源不存在                  │
  ├────────┼─────────────────────────────┤
  │ 500    │ 服务器内部错误              │

  └────────┴─────────────────────────────┘
