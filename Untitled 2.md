  1 商品管理 - 最复杂的模块

  商品列表 (搜索、筛选、批量操作)
    ├─ 商品编辑
    │  ├─ 基本信息 (名称、描述、SKU)
    │  ├─ 图片管理 (多图上传、CDN)
    │  ├─ 价格库存 (市场价、销售价、成本、库存预警)
    │  ├─ 分类标签 (分类、标签、SEO)
    │  ├─ 属性配置 (品牌、材质、尺码等)



  2 订单管理 - 最核心的模块

  订单列表 (搜索、多维度筛选、批量操作)
    ├─ 待支付订单
    │  ├─ 发送支付链接
    │  └─ 手动标记已支付
    ├─ 待发货订单
    │  ├─ 生成发货单
    │  └─ 配置物流单号
    ├─ 已发货订单



  3 用户管理 - 关键的模块

  用户列表 (搜索、标签、分层)
    ├─ 用户详情
    │  ├─ 基本信息 (昵称、头像、手机)
    │  ├─ 消费记录 (订单、金额、积分)
    │  ├─ 地址管理
